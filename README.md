## 项目简介这是一个用于管理用户和文件的后端项目，使用 FastAPI 框架构建。项目包含用户注册、登录、文件上传、文件删除和文件移动等功能。## 环境要求
- Python 3.8+
- Docker

## 安装依赖

在项目根目录下运行以下命令来安装所需的 Python 依赖：

```sh
pip install -r requirements.txt
```
